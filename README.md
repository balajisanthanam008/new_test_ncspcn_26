# NCSPCN 2026 - National Conference on Signal Processing

National Conference on Signal Processing, Communication & Networking at Sri Venkateswara College of Engineering, 8 April 2026.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
